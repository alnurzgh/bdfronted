# Smart Bookshelf - Интернет-магазин книг

Современный, минималистичный интернет-магазин книг с черно-белой цветовой схемой.

## 📋 Структура проекта

```
bdfrontend/
├── index.html              # Главная страница
├── catalog.html            # Каталог книг
├── product.html            # Страница товара
├── cart.html               # Корзина
├── profile.html            # Профиль пользователя
├── auth.html               # Авторизация/регистрация
├── css/
│   ├── reset.css           # CSS Reset
│   ├── variables.css       # Переменные дизайн-системы
│   ├── layout.css          # Общие layout стили
│   ├── components.css      # Переиспользуемые компоненты
│   └── pages/              # Стили для отдельных страниц
│       ├── home.css
│       ├── catalog.css
│       ├── product.css
│       ├── cart.css
│       ├── profile.css
│       └── auth.css
├── js/
│   ├── main.js             # Основной JavaScript
│   ├── catalog.js          # Функционал каталога
│   ├── product.js          # Функционал страницы товара
│   ├── cart-page.js        # Функционал корзины
│   ├── profile.js          # Функционал профиля
│   └── auth.js             # Функционал авторизации
└── images/                 # Изображения
```

## 🎨 Дизайн-система

### Цветовая палитра (черно-белая)
- **Primary**: #000000 (черный)
- **Background**: #FFFFFF (белый)
- **Gray Scale**: от #FAFAFA до #171717

### Типографика
- **Шрифт**: Inter (Google Fonts)
- **Размеры**: от 12px до 60px
- **Веса**: 400, 500, 600, 700

### Spacing (8px Grid System)
- Базовая единица: 8px
- Диапазон: от 4px до 96px

### Border Radius
- Small: 4px
- Medium: 8px
- Large: 12px
- XL: 16px
- 2XL: 24px
- Full: 9999px

### Shadows
- 5 уровней теней (sm, md, lg, xl, 2xl)
- Используются для карточек, модальных окон, hover эффектов

## 🚀 Функционал

### Главная страница (index.html)
- Hero секция с призывом к действию
- Категории книг (4 карточки)
- Популярные книги (grid с карточками товаров)
- Форма подписки на рассылку
- Анимации при скролле

### Каталог (catalog.html)
- Фильтры (категория, цена, автор, рейтинг)
- Сортировка (цена, популярность, новизна)
- Grid товаров (адаптивный)
- Пагинация
- Breadcrumbs

### Страница товара (product.html)
- Галерея изображений (слайдер + thumbnails)
- Информация о книге (ISBN, страницы, размер)
- Описание и об авторе
- Отзывы и рейтинг
- Похожие товары
- Выбор количества
- Кнопка "Добавить в корзину"

### Корзина (cart.html)
- Список товаров
- Изменение количества
- Удаление товаров
- Промокод
- Итоговая сумма
- Кнопка "Оформить заказ"

### Профиль (profile.html)
- Табы: Личные данные, Заказы, Избранное, Настройки
- Редактирование профиля
- История заказов
- Избранные товары
- Изменение пароля
- Настройки уведомлений

### Авторизация (auth.html)
- Форма входа
- Форма регистрации
- Восстановление пароля
- Валидация форм
- Переключение между формами
- Показ/скрытие пароля

## 📦 Установка и запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере
3. Или используйте локальный сервер:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx serve
   ```
4. Откройте http://localhost:8000
